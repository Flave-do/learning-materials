数据代理基础:通过一个对象代理对另一个对象中属性的操作(读/写)

<script type="text/javascript" >
    let obj = [x:100]
    let obj2 = [y:200)
    Object.defineProperty(obj2, 'x', {
        get(){
            return obj.x
        },
        set(value){
            obj.x= value
        }
    })
</script>
