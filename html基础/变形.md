修改视域

perspective()来修改元素的视域，它能够使得我们的3D空间产生透视的效果

参数越小视角越深

perspective（）为每个元素设置视域

perspective：父元素设置该属性，默认视域原点在中间



.one img{

​		transform: perspective(200px) rotatex(30deg)

}

.two img{ 

​		perspective: 200px;

}



perspective-origin：left top；			将视域原点设置为左上角

.two img{ 

​		perspective: 200px;

​		perspective-origin：left top；

}
