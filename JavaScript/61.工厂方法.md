使用工厂方法创建对象

通过该方法可以大比量的创建对象

```
function createPerson(){
        //创建一个新的对象
        var obj = new object();
        //向对象中添加属性
        obj.name = "孙悟空"			//写死了
        obj.age = 18;			//写死了
        obj.gender = "男";			//写死了
        obj.sayName = function(){
        		alert(this.name);
        };
        //将新的对象返回
        return obj;
var obj2 = createPerson();
var obj3 = createPerson();
var obj4 = createPerson();
```
