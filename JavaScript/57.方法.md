函数也可以称为对象的属性，

如果一个函数作为一个对象的属性保存那么我们称这个函数是这个对象的方法

调用这个函数就说调用对象的方法(method)

但是它只是名称上的区别没有其他的区别



枚举对象中的属性

使用for ...in语句

​		语法：

 			for (var  变量   in   对象){

​					}

for ...in语句对象中有几个属性,循环体就会执行几次

每次执行时,会将对象中的一个属性的名字赋值给变量

```
var obj = {
		name:"孙悟空"
		}

for(var n in obj){
		console.log ("属性名:"+n);
		console.1og("属性值:"+obj[n]);
		} 
```
