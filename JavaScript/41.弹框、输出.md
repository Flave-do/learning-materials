prompt()可以弹出一个提示框,该提示框中会带有一个文本框,

用户可以在文本框中输入一段内容,

该函数需要一个字符串作为参数,该字符串将会作为提示框的提示文字

用户输入的内容将会作为函数的返回值返回,可以定义一个变量来接收该内容

var score = prompt("请输入小明的期末成绩:");



prompt ()函数的返回值是String类型，类型转换

var num = +prompt（“请输入第一个数:”）；
